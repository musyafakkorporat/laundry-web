# 🧺 RAMA Laundry - Website Services

Website resmi **RAMA Laundry** yang berlokasi di Sukodono, Sidoarjo. Proyek ini dibangun untuk memudahkan pelanggan melihat paket harga dan melakukan simulasi pemesanan secara mandiri.

## ✨ Fitur Utama
- **Automated Price Calculation**: Menghitung total biaya berdasarkan berat (kg) dan jenis layanan (Cuci Only/Setrika/Express).
- **Data Persistence**: Menggunakan *LocalStorage* sehingga daftar pesanan tidak hilang meskipun halaman di-refresh.
- **Minimum Order Logic**: Sistem otomatis menerapkan minimal order 2kg sesuai ketentuan outlet.
- **Modern UI/UX**: Menggunakan Bootstrap 5 dengan sistem animasi *Reveal* saat scroll.
- **Interactive Map**: Integrasi lokasi outlet di Graha Asri, Sukodono.

## 🛠️ Teknologi yang Digunakan
- **Frontend**: HTML5, CSS3, Bootstrap 5
- **Logic**: Vanilla JavaScript (ES6)
- **Icons**: Bootstrap Icons
- **Animation**: Intersection Observer API

## 🚀 Cara Menjalankan
1. Clone repositori ini.
2. Buka file `index.html` di browser pilihan Anda.
3. Untuk fitur pemesanan, buka halaman `Order.html`.
